# Googleスプレッドシートで「きさきヒストリア」を運用する手順

Excelは不要です。ブラウザだけで完結します。所要時間：初回セットアップ約15分。

## 用意するもの

- Googleアカウント（Gmail等を持っていればOK。なければ [accounts.google.com](https://accounts.google.com/) で5分で作れます）
- パソコンのブラウザ（Chrome / Edge / Safari / Firefoxなど何でも）
- このフォルダ内の2つのCSVファイル
  - `01_人物.csv`（人物データ40件）
  - `02_事件解説.csv`（事件解説34件）

## ステップ1：Googleスプレッドシートを開く

1. ブラウザで [https://sheets.google.com/](https://sheets.google.com/) を開く
2. Googleアカウントでログイン（まだなら作成）
3. 「**＋ 空白**」をクリックして新しいスプレッドシートを作る
4. 左上の「無題のスプレッドシート」をクリックして「**きさきヒストリア_データ**」など好きな名前を入力

## ステップ2：人物データを取り込む

1. 開いたスプレッドシートの**シート名「シート1」**を、画面下部のタブを右クリック→「名前を変更」で「**人物**」に変更
2. メニューから「**ファイル → インポート**」をクリック
3. ダイアログで「**アップロード**」タブを選択 → `01_人物.csv` をドラッグ＆ドロップ
4. インポートの場所を「**現在のシートに追加**」または「**現在のシートを置換**」を選択
5. 区切り文字は「**カンマ**」、文字エンコードは「**UTF-8**」を確認して「**データをインポート**」
6. これで40件の人物データが表示されます

## ステップ3：事件解説シートを追加

1. 画面下部のタブの「＋」ボタンで新しいシートを追加
2. 新しいシートの名前を「**事件解説**」に変更
3. メニューの「**ファイル → インポート**」→「アップロード」→ `02_事件解説.csv`
4. インポート場所は「**現在のシートを置換**」、区切り文字「**カンマ**」、エンコード「**UTF-8**」
5. 34件の事件解説が読み込まれます

## ステップ4：見やすく整える（任意）

- 1行目を選択 → 「**表示形式 → 太字**」（Ctrl+B）でヘッダーを目立たせる
- 1行目を選択 → 「**表示 → 固定 → 1行**」でスクロール時にヘッダーを固定
- 列の幅を調整（列の境界をダブルクリックで自動調整）
- 「フィルタを作成」（漏斗アイコン）で各列に並び替え・絞り込みを付ける

## 週1の更新ワークフロー（運用開始後）

新着動画が出たら、次の手順だけです。所要時間：3〜5分。

1. ブラウザでスプレッドシートを開く
2. 「人物」シートで、最終行の下に**新しい行を追加**
3. 既存の似た行をコピペして、ID・人物名・URLなどを書き換える
4. 「関連事件」「関連キーワード」「テーマ」は**「; （セミコロン＋スペース）」区切り**で複数値を書く
5. 自動保存されるので、保存ボタンは不要

事件解説を追加したいときは「事件解説」シートに行を追加するだけです。

## 列の書き方のルール

人物シートの各列：

| 列名 | 例 | 注意点 |
|---|---|---|
| ID | `jito` | 半角英数のみ・他と重複しない |
| 人物名 | `持統天皇` | 検索でヒットする主表記 |
| 読み | `じとうてんのう` | ひらがなのふりがな |
| 動画タイトル | `持統天皇【ゆっくり解説】` | YouTubeのタイトルそのまま |
| 動画URL | `https://www.youtube.com/watch?v=...` | 動画のURL |
| 生年・没年 | `645` または `c.978` または空欄 | わかる場合のみ |
| 時代 | `飛鳥` `奈良` `平安初期` `戦国` など | 自由記述だが既存と揃える |
| 国 | `日本` `フランス` `イギリス` など | 国名 |
| 関連事件 | `壬申の乱; 藤原京遷都` | 「; 」区切りで複数 |
| 関連キーワード | `天武天皇; 万葉集` | 「; 」区切り |
| 役割 | `女帝` `皇后` `側室` など | 1〜2語 |
| テーマ | `律令国家; 女帝` | 「; 」区切り |
| 要約 | 100字程度 | 動画を見ての要約 |

## サイトへの反映方法（後日設定）

スプレッドシートのデータをサイトに取り込む方法は2通り：

**A. 半自動（おすすめの開始時）**
1. スプレッドシートで「**ファイル → ダウンロード → カンマ区切り（.csv）**」で各シートをダウンロード
2. 提供する`csv2json.py`スクリプトを実行（`python csv2json.py`）
3. 生成された`data.json`と`events.json`でサイトを更新
4. NetlifyやCloudflare Pagesに再アップロード

**B. 完全自動（運用が安定してから）**
1. スプレッドシートを「**ファイル → 共有 → ウェブに公開**」でCSV公開
2. サイトのapp.jsを修正して、起動時にCSVを直接読みに行くようにする
3. シート編集→数分後にサイト自動更新

最初はAで運用し、2〜3か月して操作に慣れたらBに移行すると安心です。

## 共有・バックアップ

- スプレッドシートは自動的にGoogle Drive内に保存され、編集履歴も自動で残ります
- 「**ファイル → 版の履歴**」で過去の状態にいつでも戻せます
- 「**共有**」ボタンで友人や咲熊さんファンの仲間と共同編集も可能（招待制）

## トラブル時

- **文字化けする** → インポート時のエンコードを「UTF-8」にする
- **「; 」が認識されない** → セルに直接「; 」を含める（区切り記号としてではなくセルの値の一部）
- **行を間違えて消した** → Ctrl+Z で戻す。または「ファイル → 版の履歴」で復元

困ったら、いつでもお声がけください。
