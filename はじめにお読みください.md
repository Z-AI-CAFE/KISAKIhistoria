# きさきヒストリア

教科書の太字キーワードから、その時代を生きた女性たちの動画にたどり着ける学習用サイトです。

**収録：人物90名（日本史40・世界史50）／事件解説60件**　古代ローマから20世紀のロシア革命まで網羅。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面の見た目・チップ・タブ |
| `app.js` | 検索ロジック・トグル |
| `data.json` | 人物データ（90名） |
| `events.json` | 事件解説データ（60件） |
| `scripts/csv2json.py` | CSV→JSON変換スクリプト |
| `scripts/json2csv.py` | JSON→CSV書き出しスクリプト |
| `scripts/週1運用ガイド.md` | 毎週の更新ワークフロー |
| `spreadsheet-template/01_人物.csv` | スプレッドシート用人物データ |
| `spreadsheet-template/02_事件解説.csv` | スプレッドシート用事件解説 |
| `spreadsheet-template/Googleスプレッドシート設定ガイド.md` | 初回セットアップ |

## 起動方法

### Webに公開する（おすすめ）

このフォルダ全体を [Cloudflare Pages](https://pages.cloudflare.com/) / [Netlify](https://app.netlify.com/) / [Tiiny Host](https://tiiny.host/) のいずれかにドラッグ＆ドロップで公開URLが取得できます。

### ローカルで確認する

VS Code の Live Server 拡張、または `python -m http.server 8000` でこのフォルダを起動 → ブラウザで `http://localhost:8000`。

`index.html` を直接ダブルクリックではブラウザのセキュリティ制約でJSONが読み込めません。必ずサーバ経由で。

## 週1更新ワークフロー

1. **初回**：`spreadsheet-template/` のCSVを Googleスプレッドシートに取り込む（`Googleスプレッドシート設定ガイド.md` 参照）
2. **毎週**：`scripts/週1運用ガイド.md` の手順に従う

要点：シートに1行追加 → CSVをダウンロード → `python scripts/csv2json.py` 実行 → フォルダ再アップロード。所要時間 約3〜5分。

## トラブル時

- **検索結果に新規追加した人物が出ない** → ブラウザでCtrl+F5でハード再読み込み
- **アップロードしてもしばらく反映されない** → CDNキャッシュ。10〜30分待つ
- **csv2json.py でエラー** → 表示されたメッセージのとおりCSVのその行を修正
- **Pythonがない** → Microsoft Storeで無料インストール
