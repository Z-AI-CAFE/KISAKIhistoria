# 週1運用ワークフロー（咲熊さんの新作1本を追加する）

このドキュメントは、咲熊さんに新作動画が公開されたら毎週やる作業を、3〜5分で終わらせるための手順です。

## 全体フロー（1週間サイクル）

```
①咲熊さんの新作公開（週1）
    ↓
②スプレッドシートを開いて1行追加（3分）
    ↓
③スプレッドシートからCSVダウンロード（30秒）
    ↓
④csv2json.py を実行（5秒）
    ↓
⑤フォルダをホスティングに再アップロード（30秒）
    ↓
⑥動作確認（30秒）→ 完了！
```

## 初回セットアップ（最初の1回のみ）

スクリプトを使うために、Python 3.7以降がPCに入っている必要があります。

**Python のインストール確認**

スタートメニューで「cmd」と入力してコマンドプロンプトを開き、`python --version` と打ってみてください。「Python 3.x.x」と表示されればOK。出ない場合は Microsoft Store で「Python」を検索して無料インストール。

**Googleスプレッドシートのセットアップ**

`spreadsheet-template/Googleスプレッドシート設定ガイド.md` の手順に従い、最新の `01_人物.csv` `02_事件解説.csv` をGoogleスプレッドシートに取り込んでください。

## 週1の作業手順（毎週これだけ）

### ステップ1：スプレッドシートに新作を追加（3分）

1. ブラウザでGoogleスプレッドシートの「人物」シートを開く
2. 一番下の行にカーソルを置き、Enterキーで新しい行に移動
3. 既存の似た時代・国の行を選択してCtrl+Cでコピー、新しい行にCtrl+Vで貼り付け（書式を引き継ぐため）
4. 各セルを書き換え：
   - **ID** ：人物名のローマ字（例：`takeko` `marie_louise2`）。他と被らないユニークな短い英数
   - **人物名** ：表示用の名前（例：`徳子内親王`）
   - **読み** ：ふりがな（例：`とくこないしんのう`）
   - **動画タイトル** ：YouTubeのタイトルそのまま
   - **動画URL** ：動画ページのURL（`https://www.youtube.com/watch?v=...`）
   - **生年・没年** ：分かれば数字。不確かなら `c.1500` `1500?` のように
   - **時代** ：`平安時代中期` など、既存の表記を真似する
   - **国** ：`日本` `フランス` `イングランド` などシンプルに
   - **関連事件** ：教科書的な事件を `; ` 区切り（例：`応仁の乱; 明応の政変`）
   - **関連キーワード** ：人物・地名・関連語を `; ` 区切り
   - **役割** ：`皇后` `側室` `女御` など短く
   - **テーマ** ：`摂関政治; 宮廷文化` のような分類タグ
   - **要約** ：100〜150字程度の概要
5. 自動保存されるので保存ボタン不要

新しい事件を解説したい場合は「事件解説」シートにも1行追加。

### ステップ2：CSVをダウンロード（30秒）

スプレッドシート画面で：

1. 画面下部のタブで「人物」シートを選択
2. メニュー「**ファイル → ダウンロード → カンマ区切り（.csv）**」
3. ダウンロードフォルダに `人物.csv` のようなファイルが保存される
4. ファイル名を **`01_人物.csv`** にリネームして、サイトフォルダの `spreadsheet-template/` に上書き保存
5. 「事件解説」シートも同様に `02_事件解説.csv` として保存

### ステップ3：JSONに変換（5秒）

サイトフォルダで右クリック→「ターミナルで開く」もしくはコマンドプロンプトで：

```
python scripts/csv2json.py
```

エラーがなければ `data.json` と `events.json` が更新されます。

```
=== JSON書き出し ===
  ✅ /path/to/data.json
  ✅ /path/to/events.json

📊 完了：人物 91名（日本41 / 世界50）／事件解説 60件
```

エラーが出た場合は、どの行のどの項目に問題があるか表示されるので修正してください。

### ステップ4：再アップロード（30秒）

**Netlify** をお使いなら：
1. Netlify管理画面のサイトを開く
2. 「**Deploys**」タブ
3. 一番下にある「Drag and drop your site folder」エリアに、サイトフォルダ全体をドロップ
4. 数秒待つと新しいデプロイが完了

**Cloudflare Pages** や他のサービスも同様、フォルダ全体を再アップロード。

### ステップ5：動作確認（30秒）

1. サイトURLを開く（CDNキャッシュ回避のため Ctrl+F5 でハード再読み込み）
2. 検索ボックスに新しく追加した人物名を入れて、ヒットすることを確認
3. 関連事件のチップをクリックして、新しい人物が現れることを確認

完了！ 所要時間は慣れれば5分以内です。

## 週1運用のコツ

**コツ1：行のコピー＆ペーストを活用**

ゼロから新規行を入力すると時間がかかるので、似た時代・国の既存行をコピーして、それぞれのセルを書き換えるのが速いです。

**コツ2：分からない項目は空欄でOK**

生没年や役割が不明な場合は空欄のままで構いません。後から埋められます。

**コツ3：要約は動画を見ながら書く**

咲熊さんの動画を見ながらメモを取り、最後に要約を書くと自然です。

**コツ4：エラーが出ても焦らない**

`csv2json.py` がエラーを出すのは、データの不整合を未然に防ぐためです。表示されたメッセージに従って修正すれば必ず動きます。

## トラブル時の対処

**「python が認識されません」**

Pythonがインストールされていません。Microsoft Storeから「Python」を無料インストールし、cmdを開き直して再実行。

**「文字化けしている」**

CSVをダウンロードした時の文字コードが UTF-8 になっていない可能性があります。Googleスプレッドシートのダウンロードは標準でUTF-8なので通常は問題ありません。

**「サイトに反映されない」**

ブラウザのキャッシュです。Ctrl+F5（Mac は Cmd+Shift+R）でハード再読み込み。それでもダメなら10〜30分待ってから再度試す（CDNキャッシュ）。

**「ID重複エラーが出る」**

`csv2json.py` のメッセージに従い、重複しているIDをユニークなものに変更してください。

**「変なところに文字化けが…」**

セルに `"` （ダブルクォート）が入っていると壊れる場合があります。CSVは `"` を文字列の区切りとして使うので、本文中に使う場合はエスケープ（`""` のように2つ重ねる）が必要です。Googleスプレッドシートが自動でやってくれます。

## バックアップ運用

JSONを直接編集してしまった場合や、サイトをローカルで触ってシートと不整合が生じた場合は、`json2csv.py` でJSONから最新CSVを書き戻せます：

```
python scripts/json2csv.py
```

これで `spreadsheet-template/` 内のCSVが最新JSONと一致するようになります。その後、Googleスプレッドシートを「ファイル → インポート → 現在のシートを置換」で更新してください。
